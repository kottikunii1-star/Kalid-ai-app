# Kalid AI App - Afaan Oromoo AI Chat

Kalid AI Pro - Afaan Oromoo irratti hojjechaa jira. Express, OpenAI API, fi Vercel-tti deploy goomuun hojjechaa.

## Fashaloonni

- ✅ Afaan Oromoo chat interface
- ✅ OpenAI GPT-4o-mini integration
- ✅ Express server
- ✅ CORS support
- ✅ Vercel deployment ready

## Installation

```bash
npm install
```

## Environment Variables

`.env` file keessatti dabaluu:

```
OPENAI_API_KEY=your_api_key_here
PORT=5000
```

## Local Development

```bash
npm start
```

Server http://localhost:5000 irratti hojjechaa.

## Deployment

Vercel irratti deploy goodhuf:

1. https://vercel.com irratti login gochuu
2. GitHub repository import gochuu
3. Environment variables dabaluu (OPENAI_API_KEY)
4. Deploy qoqodduu

## Faayiloonni

- `index.js` - Main server file
- `package.json` - Dependencies
- `vercel.json` - Vercel configuration
